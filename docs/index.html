circularShift

Rotates an array

circularShift(array: Array<any>, count: number): Array<any>
Parameters
array (Array<any>)
count (number)
Returns
Array<any>
Example
// returns [2, 3, 1]
circularShift([1, 2, 3], 1);
// returns [3, 1, 2]
circularShift([1, 2, 3], -1);
// returns [1, 2, 3]
circularShift([1, 2, 3], 0);
// returns [1, 2, 3]
circularShift([1, 2, 3], 6);
// returns [2, 3, 1]
circularShift([1, 2, 3], -3);
